# Claude Code Guide Skill

帮助 OpenClaw 更好地调用 Claude Code Skills 的指南。

## 包含内容

- Skill 格式和结构
- 调用方式和最佳实践
- 创建新 Skill 的步骤
- 推送到 GitHub 的方法

## 使用方法

当需要创建或使用 Claude Code Skills 时，调用此 skill:
```
/claude-code-guide
```
